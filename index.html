Generalidades y utilidades del CSS Grid

Grid y sus generalidades

Una cuadrícula o "grid" se define como un conjunto de líneas horizontales y verticales que delimitan espacios específicos uno a los lados de los otros, formando filas y columnas.

Origen del Grid en CSS

Una de las principales dificultades por las que atraviesan los programadores, principalmente principiantes, es la asignación de espacios dentro del sitio que se está diseñando, y aunque existen diversas herramientas para solucionar este problema, suelen ser ineficientes o en extremo complejas.
El origen de Grid CSS se encuentra en la necesidad de solucionar este problema, añadiendo a un sistema grid primitivo en ciertas características, ventajas y mejoras extensas que permiten crear cuadrículas flexibles de forma rápida.

Creacion de diseños basados en Grid CSS

Para llevar a cabo el diseño eficiente de cuadrículas basadas en Grid CSS se requiere el manejo de ciertos conceptos básicos, los cuáles son:

  • Contenedor: Elemento que define la rejilla.
  • Ítem: Cada una de las regiones que constituyen la rejilla.
  • Celda: Unidad mínima de la cuadrícula.
  • Área: Una región o agrupación de celdas.
  • Banda: Grupo de celdas de forma horizontal o vertical en la cuadrícula.
  • Línea: Separadores y limitadores de las regiones.
